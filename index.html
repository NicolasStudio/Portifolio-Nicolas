Já a longo prazo, meu sonho é conquistar minha casa própria e me tornar uma referência no mundo da programação.

Gerador de Dados

Este sistema foi desenvolvido para me auxiliar — e também a outras pessoas — na criação de dados fictícios, porém válidos, destinados a testes de cadastros em sistemas. Ele é capaz de gerar automaticamente nome completo, CPF, RG, data de nascimento, telefone fixo, celular e e-mail, todos devidamente formatados para facilitar seu uso em ambientes de desenvolvimento. Além disso, inclui um gerador de senhas aleatórias e, mais recentemente, passou a contar com um contador de caracteres, oferecendo ainda mais praticidade no dia a dia dos testes.

Você pode conferir através do Github.

Pizzaria

Esta página foi desenvolvida como parte do meu aprendizado, com o objetivo de praticar novas técnicas de desenvolvimento. Trata-se de uma interface para uma pizzaria, que conta com um menu com navegação por ancoragem, permitindo que o usuário explore o cardápio de forma intuitiva. O menu é interativo: ao selecionar os itens, os pedidos são adicionados automaticamente ao carrinho. Um dos destaques é a opção de montagem personalizada, onde o próprio cliente pode escolher até quatro sabores de pizza, que são adicionados ao carrinho após a confirmação. Ao finalizar o pedido, o sistema encaminha automaticamente as informações para a pizzaria via WhatsApp.

Você pode conferir através do Github.

Landing page

Este foi o meu primeiro trabalho como desenvolvedor web freelancer, onde tive a oportunidade de entender as necessidades do cliente e criar uma landing page para divulgação de seus serviços. Foi um projeto desafiador, especialmente pela necessidade de implementar uma galeria com carrossel de imagens. A página conta com navegação por ancoragem nas seções de início, sobre, habilidades, projetos e contatos. Também foi incluído um sistema de pop-ups para facilitar a visualização das etapas de construção, proporcionando uma experiência mais intuitiva para os usuários.

Você pode conferir através do Github.

Livraria

Esta é uma página voltada para a venda de e-books, com sistema de cadastro de usuários e funcionalidades completas de gerenciamento de livros, incluindo inclusão, edição e exclusão. O layout conta com menus organizados em carrossel, integrados a um banco de dados para garantir dinamismo e atualização em tempo real. Além disso, a plataforma permite a atualização de dados cadastrais tanto para usuários convencionais quanto para administradores. O sistema também já conta com uma forma de pagamento integrada, oferecendo uma experiência completa e funcional.

Versão com PHP e Banco de dados no Github.

Zer0Kcalc

Este sistema foi desenvolvido para oferecer informações precisas sobre gasto calórico e percentual de gordura corporal. Na tela inicial, o usuário encontra duas ferramentas principais: a Calculadora de Percentual de Gordura, que utiliza o método das dobras cutâneas para medir a quantidade de gordura corporal por meio da análise de três ou sete pregas de pele em áreas específicas do corpo; e a Calculadora de Gasto Calórico, que realiza o cálculo da necessidade diária de calorias, indicando quanto o usuário deve consumir para ganhar massa muscular ou reduzir a ingestão calórica para perder peso.

Você pode conferir através do Github.
